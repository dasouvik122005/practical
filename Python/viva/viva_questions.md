# Python Viva Questions

## What is Python?
Python is a high-level, interpreted, interactive and object-oriented scripting language. Python is designed to be highly readable. It uses English keywords frequently where other languages use punctuation, and it has fewer syntactical constructions than other languages.

## What are the key features of Python?
- **Easy to learn and use:** Python has a simple syntax.
- **Interpreted Language:** Python is processed at runtime by the interpreter. You do not need to compile your program before executing it.
- **Cross-platform:** Python can run on various platforms like Windows, macOS, and Linux.
- **Expressive Language:** Python language is more expressive as it allows you to write code more concisely.
- **Large Standard Library:** Python provides a vast standard library that has some handy codes and functions which we can use to build our applications.
- **Object-Oriented Language:** Python supports object-oriented language and concepts of classes, objects encapsulation, etc.
- **Extensible:** It implies that other languages such as C/C++ can be used to compile the code and thus it can be used further in our python code.

## What is an interpreted language?
An interpreted language is a type of programming language for which most of its implementations execute instructions directly and freely, without previously compiling a program into machine-language instructions.

## What is a variable in Python?
A variable is a named location used to store data in the memory. It is helpful to think of variables as a container that holds data that can be changed later in the program.

## What are Python data types?
- **Numbers:** (integer, float, complex)
- **String:** (str)
- **Sequence Type:** (list, tuple, range)
- **Mapping Type:** (dict)
- **Set Types:** (set, frozenset)
- **Boolean Type:** (bool)
- **Binary Types:** (bytes, bytearray, memoryview)

## What is a list in Python?
A list is an ordered and mutable collection of items. Lists are defined by having values between square brackets `[]`.

## What is the difference between a list and a tuple?
The main difference is that lists are mutable, and tuples are immutable. This means lists can be changed, and tuples cannot be.

## What is a dictionary in Python?
A dictionary is an unordered collection of data values, used to store data values like a map, which, unlike other Data Types that hold only a single value as an element, Dictionary holds key:value pair.

## What is indentation in Python?
Indentation refers to the spaces at the beginning of a code line. Where in other programming languages the indentation in code is for readability only, the indentation in Python is very important. Python uses indentation to indicate a block of code.

## What is a function in Python?
A function is a block of organized, reusable code that is used to perform a single, related action.

## What is the difference between == and is?
- `==` is for value equality. It's used to know if two objects have the same value.
- `is` is for reference equality. It's used to know if two references refer to the same object.

## What is a loop in Python?
A loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

## What is the difference between for loop and while loop?
- **for loop:** It is used to iterate over a sequence of items.
- **while loop:** It is used to execute a block of statements repeatedly until a given condition is satisfied.

## What is exception handling in Python?
Exception handling is the process of responding to the occurrence of exceptions – anomalous or exceptional conditions requiring special processing – during the execution of a program. In Python, you can use `try...except` blocks to handle exceptions.

## What is a module in Python?
A module is a file containing Python definitions and statements. The file name is the module name with the suffix `.py` appended. We use modules to break down large programs into small manageable and organized files.
